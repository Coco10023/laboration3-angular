# Laboration 3 - Angular

## Publicerad webbplats
Länk: https://coco10023.github.io/laboration3-angular/ 

## Beskrivning
Detta projekt är en webbplats byggd med Angular. Syftet med uppgiften var att
lära sig grunderna i Angular, hur komponenter fungerar och hur man använder routing
för att skapa en dynamisk webbapplikation.

## Funktioner
- Startsida
- Konverterarsida
- Om-sida
- Navigering med Angular Router
- Konvertering mellan:
  - meter och feet
  - celsius och fahrenheit
- En återanvändbar komponent (info-card)
- Responsiv design
- Bild på startsidan

## Tekniker
- Angular
- TypeScript
- HTML & CSS
- Git & GitHub

## Installation
```bash
npm install
ng serve
```

## Reflektion

Under arbetet har jag lärt mig hur Angular fungerar, hur man bygger upp en applikation
med komponenter och hur routing används. Jag har även fått förståelse för databindning
genom att skapa en konverterare.

En utmaning var att förstå filstrukturen i Angular och hur olika delar hänger ihop,
men genom att testa och felsöka kunde jag lösa problemen.
